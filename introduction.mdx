## Introduction
Truemetrics Solution consits of a SDK to record sensor data in the drivers app and an RESTAPI allowing to query insights about the delivery process and to improve the last mile by 
providing high resolution location data, like entrances and parking positions.


## SDK enabled features

Truemetrics Android SDK provides an easy way to collect the positional & motion data of the devices and convert them into intelligent insights to solve many challenges in the last mile delivery likes finding correct entrances and parking places but also getting deep delivery insights.
